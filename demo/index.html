CirclesSparkline WEB COMPONENT

<circles-sparkline/> is a web component that can be used in any website (you can use it in a pure JS, React, Angular, or VueJS website)

Users can input a set of numbers and customize the sparkline width, height, and stroke width:

<circles-sparkline width="300" height="50" id="sample1" stroke-width="3" data="[10,20,5,15,2,35]" />

TECH DETAILS

Although this component was created using Angular, I thought it would be more flexible and useful if I export it as a web component.

In order to build a new package of this component, follow the steps:

  1. Open a terminal and go to the path where package.json exists
  2. If node_modules is not existing, run npm run install
  3. Run npm run build
  4. After a successful build, go to the dist/halo folder and there should be three javascript files inside
    • main.xxxxxx.js
    • polyfills.xxxxxx.js
    • runtime.xxxxxx.js
  5. Use the three javascript in any websites and just call <circles-sparkline/> as seen in this demo
